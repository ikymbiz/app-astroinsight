# AstroInsight

## 概要・説明
**AstroInsight** は、最新のAI（Gemini 2.5 Flash）を活用した、ポジティブでポップな占星術＆自己分析Webアプリケーションです。

ユーザーの出生情報（ネイタル）を基に、AIが親友のように親身になってあなたの強みや魅力を分析します。日々の星の動き（トランジット）が与える影響の解説、1年半の未来予測グラフ、さらにはユーザーの情報を記憶した専属AI占い師とのチャット機能まで備えた、本格的かつ自己肯定感を高めるためのパーソナル占星術ツールです。

完全なクライアントサイド（ブラウザ内）で動作し、APIキーや入力データはすべて端末内のIndexedDBにセキュアに保存されるため、プライバシーも守られます。PWA（Progressive Web App）にも対応しており、スマートフォンからネイティブアプリのように快適に利用できます。

## 主な機能一覧
- 👤 **プロフィール管理**
  - 名前、生年月日、出生時間、出生地を登録・保存。
- ✨ **ポジティブ自己分析**
  - 入力されたプロフィールから、AIがあなたの「最強の強み」「隠れた魅力」を分析し、自己肯定感を爆上げするメッセージを生成します。（Markdown形式対応）
- ☄️ **トランジット（星の動き）分析**
  - 指定した日時における星の配置が、あなたの出生図にどのような影響を与えるかを本格的に解説。過去の分析結果の履歴管理も可能です。
- 📈 **星のタイムライン（未来予測）**
  - 今月から18ヶ月間の運勢スコアとアドバイスをAIが予測し、Chart.jsを用いた折れ線グラフと、スワイプ可能なカードUIで直感的に表示します。
- 💬 **専属AI占い師チャット**
  - あなたのプロフィールや過去の分析結果を記憶したAIと、チャット形式でいつでも悩みを相談できます。
- ⚙️ **セキュアな設定とバックアップ**
  - **Gemini APIキーのローカル保存**: サーバーを介さず直接GoogleのAPIと通信します。
  - **5世代自動バックアップ**: ローカルストレージ（IndexedDB）を活用したデータの自動保存と、JSON形式でのエクスポート/インポート・復元機能を搭載。
- 📱 **快適なUX/UI**
  - Tailwind CSSによるモダンでクリーンなモバイルファーストデザイン。
  - バックグラウンド処理（Web Worker）とストリーミング表示により、AIの長文生成中も画面がフリーズしません。
  - 画面スリープ防止機能（Wake Lock API）を搭載し、生成中の画面消灯を防ぎます。

## 使用技術・ライブラリ
- **フロントエンド**
  - HTML5 / CSS3 / JavaScript (Vanilla)
  - [Tailwind CSS](https://tailwindcss.com/) (CDN) - スタイリング
  - [FontAwesome 6](https://fontawesome.com/) (CDN) - アイコン
  - [Marked.js](https://marked.js.org/) (CDN) - Markdownパース
  - [Chart.js](https://www.chartjs.org/) (CDN) - タイムライングラフ描画
- **Web API & ブラウザ機能**
  - [Google Gemini API](https://aistudio.google.com/) (`gemini-2.5-flash`) - AIテキスト生成（SSEによるストリーミング処理）
  - IndexedDB - データのローカル保存・履歴管理
  - Service Worker - PWA対応、キャッシュ制御
  - Web Worker - AI通信・データ処理の非同期化（UIブロック防止）
  - Wake Lock API - 処理中の画面スリープ防止

## セットアップ・使い方

本アプリは静的なHTMLファイル一つで構成されているため、環境構築は不要です。

### 1. アプリの起動
提供されたHTMLファイルをダウンロードし、Google Chrome や Safari などのモダンブラウザで開きます。
（または、GitHub Pages、Vercel、Netlifyなどの静的ホスティングサービスにデプロイしてアクセスしてください）

### 2. APIキーの設定
1. [Google AI Studio](https://aistudio.google.com/) にアクセスし、無料の **Gemini APIキー** を取得します。
2. アプリ画面右上の「歯車アイコン（設定タブ）」を開きます。
3. 「API設定」の入力欄に取得したAPIキーを貼り付け、「APIキーを保存・更新」ボタンを押します。

### 3. プロフィールの入力
1. 画面下部のナビゲーションから「プロフ」タブを開きます。
2. あなたの「名前」「生年月日」「出生時間」「出生地」を入力し、「保存して分析へ」を押します。

### 4. アプリを楽しむ
- **分析タブ**: 「生成・更新」ボタンを押すと、AIによる自己分析がスタートします。
- **トランジットタブ**: 見たい日付（と時間）を指定し、「星の動きを分析」ボタンを押すと、その日の運勢と星の配置について解説されます。
- **未来予測タブ**: 「予測開始」ボタンを押すと、今月から1年半分の運勢グラフと月ごとのアドバイスが生成されます。
- **チャットタブ**: AI占い師に、恋愛、仕事、人生の悩みなど、何でも気軽にチャットで質問してみてください。プロフィールと分析結果を踏まえた専用のアドバイスがもらえます。

### 💡 データのバックアップについて
「設定タブ」から、現在までの全履歴（分析結果、チャット履歴、設定内容など）をJSONファイルとしてダウンロードしたり、別のブラウザへインポートしてデータを引き継ぐことが可能です。
