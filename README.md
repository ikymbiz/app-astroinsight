# AstroInsight

## 概要・説明
**AstroInsight** は、Google Gemini APIを活用したモバイルファーストの「AI占星術＆自己分析」Webアプリケーションです。
ユーザーの出生情報（ネイタル）を元に、AIがポジティブでポップな自己分析や未来の運勢予測、特定日時のトランジット分析を行います。

すべてのユーザーデータおよびAPIキーはブラウザのローカルストレージ（IndexedDB）に安全に保存され、外部サーバーを介さずに直接AIと通信します。また、Web Workerを利用したストリーミング生成や、Service WorkerによるPWA対応、Wake Lock APIによる画面スリープ防止など、モダンなWeb技術を盛り込んだ快適なユーザー体験を提供します。

## 主な機能一覧
- **🧑‍🚀 プロフィール管理**
  - 名前、生年月日、出生時間、出生地を入力し、ローカルに自動保存。
- **✨ AI自己分析（ネイタル）**
  - 入力されたプロフィールから、ユーザーの「最強の強み」や「隠れた魅力」をAIがポジティブなトーンで分析。
  - ストリーミング表示に対応し、途中で画面を閉じても再開可能なレジューム機能を搭載。
- **☄️ 星の動き（トランジット分析）**
  - 任意の日付・時間を指定し、その時の星の配置が自身の運勢にどう影響するかを本格的に分析。
  - 過去に生成したトランジット分析の履歴保存・切り替え機能。
- **📈 星のタイムライン（未来予測）**
  - 今月から1年半（18ヶ月間）の運勢をAIが予測し、Chart.jsを用いた影響度グラフと、スワイプ可能な月別アドバイスカードで視覚的に表示。
- **💬 星占い師AIチャット**
  - ユーザーのプロフィール情報や自己分析の結果を記憶したAI占い師と、チャット形式で具体的な悩み相談が可能。
- **⚙️ 設定・高度なデータ管理**
  - Gemini APIキーのセキュアなローカル保存機能。
  - 分析データやチャット履歴のJSON形式でのエクスポート（ダウンロード）＆インポート（復元）。
  - 自動5世代バックアップと、直近バックアップからのワンクリック復元機能。

## 使用技術・ライブラリ
本プロジェクトはビルドツールを使用せず、1つのHTMLファイルで動作するSPA（Single Page Application）として構成されています。

### フロントエンド / UI
- **HTML5 / CSS3 / Vanilla JavaScript**
- **Tailwind CSS** (CDN) - モバイルフレンドリーなモダンUIの構築
- **Font Awesome 6** (CDN) - アイコンの表示
- **Chart.js** (CDN) - タイムラインの運勢グラフ描画
- **marked.js** (CDN) - AIが生成したMarkdownテキストのパース・HTML変換

### ブラウザAPI / 非同期処理
- **IndexedDB** - データ、履歴、APIキーのローカル永続化
- **Web Worker** - UIスレッドをブロックしないバックグラウンドでのAIストリーミング通信
- **Service Worker** - PWA（Progressive Web App）キャッシュ対応
- **Wake Lock API** - AI分析中の画面スリープ防止

### AI / 外部API
- **Google Gemini 2.5 Flash API** (Server-Sent Events: SSE 形式でのストリーミング生成)

## セットアップ・使い方

### 前提条件
本アプリを利用するには、Google AI Studioにて発行された **Gemini APIキー** が必要です。
事前に [Google AI Studio](https://aistudio.google.com/) にアクセスし、APIキーを取得してください。

### セットアップ
1. リポジトリのソースコード（HTMLファイル）をローカルにダウンロード、またはクローンします。
2. ブラウザで `index.html` を直接開くか、ローカルサーバー（VS CodeのLive Serverなど）経由で開きます。
   *※Service WorkerやWake Lock APIの機能を完全に動作させるためには、ローカルサーバー経由（またはHTTPS環境）での実行を推奨します。*

### 使い方
1. **APIキーの設定**
   - 画面右上の歯車アイコン（⚙️）、または下部ナビゲーションから「設定」タブを開きます。
   - 取得したGemini APIキーを入力し、「APIキーを保存・更新」ボタンを押します。
2. **プロフィールの入力**
   - 下部ナビゲーションの「プロフ」タブを開き、あなたの情報を入力して「保存して分析へ」を押します。
3. **自己分析の生成**
   - 「分析」タブに移動し、右上の「生成・更新」ボタンを押すと、AIがストリーミング形式で自己分析結果を出力します。
4. **ホロスコープ（トランジット）の確認**
   - 「トランジット」タブで気になる日付を指定して分析ボタンを押すと、その日の星の配置とあなたへの影響を解説します。
5. **未来予測の確認**
   - 「未来予測」タブで「予測開始」ボタンを押すと、今後18ヶ月分の運勢グラフと月別のアドバイスが生成されます。
6. **AI占い師に相談**
   - 「チャット」タブでは、ここまでの分析結果を踏まえたAI占い師に、自由に質問や相談をすることができます。
