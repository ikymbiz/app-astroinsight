# AstroInsight

## 概要・説明
**AstroInsight** は、最新の生成AI（Gemini 2.5 Flash）を活用したパーソナル占星術＆自己分析Webアプリケーションです。

ユーザーのプロフィール（生年月日、出生時間、出生地）をもとに、AIがホロスコープ（星の配置）を推測・計算し、個人の「強み」「課題」「潜在能力」を深く分析します。また、今後1年間の星の運行（トランジット）の影響をグラフ化したり、自分専用の星占い師AIとチャットで相談したりすることが可能です。

本アプリは単一のHTMLファイル（SPA）として構築されており、バックエンドサーバーを必要としません。すべてのデータとAPIキーはブラウザのローカルデータベース（IndexedDB）に安全に保存され、PWA（Progressive Web App）としても動作するモダンな設計となっています。

## 主な機能一覧

- 👤 **プロフィール管理**
  - 名前、生年月日、出生時間、出生地の入力。
  - 入力内容はオートセーブでローカルに自動保存されます。
- ✨ **AI深層自己分析**
  - 入力されたプロフィールから、AIが専門的な占星術的観点でホロスコープを読み解き、励ましに満ちたトーンで自己分析レポートを生成します。
- 📈 **星のタイムライン（未来予測）**
  - 今後1年間の星の動きが与える影響度を0〜100のスコアで算出。
  - Chart.jsによる美しい「影響度グラフ」と、月ごとの具体的なテーマ・解説をJSONフォーマット経由で生成・表示します。
- 💬 **星占い師AIチャット**
  - プロフィール情報と生成済みの自己分析データを記憶したAI占い師と、対話形式で個別の悩みや運勢について相談できます。
- ⚙️ **高度なデータ管理と設定**
  - Gemini APIキーの安全なローカル保存。
  - 最大5世代のローカル自動バックアップ機能。
  - JSON形式でのデータのエクスポート（ダウンロード）およびインポート（復元）機能。
- 🚀 **バックグラウンド処理**
  - Web Workerを利用した非同期通信により、AIの長文生成中もUIがフリーズしません。
  - WakeLock APIにより、生成中のスマートフォンの画面スリープを防止します。
  - 万が一生成が中断されても、レジューム機能で途中から再開可能です。

## 使用技術・ライブラリ

**フロントエンド・UI**
- HTML5 / CSS3 / Vanilla JavaScript (SPAアーキテクチャ)
- **Tailwind CSS** (CDN) - モダンでレスポンシブなスタイリング
- **FontAwesome** (CDN) - アイコン表示

**データ可視化・テキスト処理**
- **Chart.js** (CDN) - タイムラインの折れ線グラフ描画
- **Marked.js** (CDN) - AIが生成したMarkdownテキストのHTMLパース

**AI・Web API**
- **Google Gemini 2.5 Flash API** (REST API / ストリーミング)
- **IndexedDB** - ユーザーデータ、チャット履歴、バックアップのローカル保存
- **Web Workers** - メインスレッドをブロックしない非同期AI通信
- **PWA (Service Worker / Manifest)** - オフラインキャッシュ（アプリシェル）とホーム画面追加対応
- **WakeLock API** - 処理中の画面消灯防止

## セットアップ・使い方

本アプリはフロントエンドのみで完結しているため、複雑な環境構築は不要です。

### 1. 準備：Gemini APIキーの取得
本アプリを利用するには Google の Gemini API キーが必要です。
1. [Google AI Studio](https://aistudio.google.com/) にアクセスし、Googleアカウントでログインします。
2. 「Get API key」から新しいAPIキーを作成し、コピーしておきます。

### 2. アプリの起動
提供されたHTMLファイルを任意のWebサーバーでホスティングするか、ローカル環境で起動します。
- **VS Codeを使用する場合**: 拡張機能「Live Server」をインストールし、HTMLファイルを右クリックして「Open with Live Server」を選択します。
- **ブラウザで直接開く**: ファイル（`index.html`）をブラウザにドラッグ＆ドロップするだけでも動作しますが、Web Workerやモジュールの仕様上、ローカルサーバー経由での起動を推奨します。

### 3. アプリの初期設定
1. アプリが起動したら、右上の「歯車アイコン（設定）」を開きます。
2. 「API設定」の入力欄に、取得したGemini APIキーを貼り付け、「APIキーを保存・更新」ボタンを押します。
   *(※キーはブラウザのIndexedDB内にのみ保存され、外部サーバーには送信されません)*

### 4. 使い方
1. **プロフタブ**: あなたの名前、生年月日、出生時間、出生地を入力します。（入力すると自動で保存されます）
2. **自己分析タブ**: 「生成・更新」ボタンを押すと、AIがホロスコープの分析を開始します。画面を切り替えても裏で処理は継続されます。
3. **タイムラインタブ**: 「予測開始」ボタンを押すと、今後1年間の運勢グラフと月ごとのアドバイスが生成されます。
4. **AIチャットタブ**: AIがあなたの分析結果を踏まえた上で待機しています。気になることや具体的な悩みについて、自由にメッセージを送信してください。
